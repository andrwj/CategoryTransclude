# CategoryTransclude 파서 함수 통합 테스트 시나리오
# 미디어위키의 파서 테스트 러너(ParserTests)를 통해 실행할 수 있습니다.

!! test
기본적인 카테고리 삽입 파서 함수 호출 (멤버가 없을 때 빈 목록 메시지 출력 검증)
!! wikitext
{{#categorytransclude:KontexusNote|limit=2}}
!! html/php
<div class="categorytransclude-empty">카테고리 &quot;분류:KontexusNote&quot;에 조건에 맞는 멤버 문서가 없습니다.</div>
!! end

!! test
한국어 매직 워드 별칭 호출 테스트
!! wikitext
{{#분류삽입:KontexusNote|limit=2}}
!! html/php
<div class="categorytransclude-empty">카테고리 &quot;분류:KontexusNote&quot;에 조건에 맞는 멤버 문서가 없습니다.</div>
!! end

!! test
잘못된 카테고리 이름 입력 에러 메시지 노출 검증 (show-errors=1)
!! wikitext
{{#categorytransclude:Invalid<Title>Name|show-errors=1}}
!! html/php
<span class="error">오류: 잘못된 카테고리 이름 &quot;Invalid&lt;Title&gt;Name&quot;입니다.</span>
!! end
